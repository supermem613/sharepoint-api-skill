## Task

Create a new folder named "EVAL_TEST_Folder" in the default document library (Shared Documents). After creation, list the library's root contents and confirm the folder appears.

## Checks

- [ ] Used sp-post.js to create the folder via /_api/web/folders or /add endpoint
- [ ] Folder was created with the name "EVAL_TEST_Folder"
- [ ] Verified the folder exists by listing the library contents
- [ ] Output shows the folder in the directory listing
