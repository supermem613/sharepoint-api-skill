## Task

Pick any list on this site that contains items. Retrieve the first 5 items from that list using `$top=5`. Display each item's Id and Title.

## Checks

- [ ] Used sp-get.js to make the API call
- [ ] Called /items endpoint with $top=5 query parameter
- [ ] Output shows up to 5 items
- [ ] Each item in the output includes its Id
- [ ] Each item in the output includes its Title
