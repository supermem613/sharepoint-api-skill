## Task

List all files in the default document library (typically "Shared Documents" or "Documents"). For each file, display the filename (Name) and size (Length or Size).

## Checks

- [ ] Used sp-get.js or graph-get.js to make the API call
- [ ] Called a Files endpoint (e.g., /Folders('Shared Documents')/Files) or Graph drives/items/children
- [ ] Output shows file names
- [ ] Output shows file sizes
