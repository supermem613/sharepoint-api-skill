## Task

Get the current authenticated user's profile information including display name, email address, and user ID.

## Checks

- [ ] Used sp-get.js with /_api/web/currentuser OR graph-get.js with /me
- [ ] Output shows the user's display name (Title or displayName)
- [ ] Output shows the user's email address (Email or mail)
- [ ] Output shows the user's ID (Id or id)
