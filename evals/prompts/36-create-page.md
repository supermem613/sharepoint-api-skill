## Task

Create a new modern site page named "EVAL_TEST_Page" with a title of "Eval Test Page". Publish the page after creation. Verify it appears in the Site Pages library.

## Checks

- [ ] Used sp-post.js to call /_api/sitepages/pages or the Site Pages list endpoint
- [ ] POST body includes the page name and title
- [ ] Published the page after creation (called publish endpoint or set publishing status)
- [ ] Verified the page appears in Site Pages by listing pages
