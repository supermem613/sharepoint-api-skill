## Task

Retrieve the site's top navigation bar (global navigation) and quick launch (left navigation) nodes. For each node, display the Title and Url.

## Checks

- [ ] Used sp-get.js to call /_api/web/navigation/topnavigationbar and/or quicklaunch
- [ ] Output includes at least one navigation node
- [ ] Each node shows Title
- [ ] Each node shows Url
