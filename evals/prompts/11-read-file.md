## Task

Read the content of a text-based file (e.g., .txt, .md, .csv) from a document library on this site. Display the file's text content.

## Checks

- [ ] Used sp-get.js with /$value endpoint OR graph-get.js with /content
- [ ] Targeted a specific file by name or path
- [ ] Output displays the actual text content of the file
