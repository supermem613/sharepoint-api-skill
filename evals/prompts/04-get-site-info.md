## Task

Retrieve the current SharePoint site's metadata including its Title, URL, and Description.

## Checks

- [ ] Used sp-get.js to make the API call
- [ ] Called /_api/web
- [ ] Output shows the site Title
- [ ] Output shows the site Url
- [ ] Output shows the site Description (even if empty)
