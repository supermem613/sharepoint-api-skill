## Task

Discover the first non-hidden list on this site, then retrieve its full schema (all columns/fields). For each field, show the Title, InternalName, and TypeAsString.

## Checks

- [ ] Used sp-get.js to make the API calls
- [ ] Called a /fields endpoint on the selected list
- [ ] Output includes field Title for each column
- [ ] Output includes InternalName for each column
- [ ] Output includes TypeAsString for each column
